# SpotLight

## About

A geolocation-based platform for sharing and discovering posts tied to places around you.

## How To Build

### For the backend:
* ```cd backend```
* ```go build -o ./server.exe ./src```
* ```./server.exe```

### For the frontend:
* ```cd frontend```
* ```npm install```
* ```npm run dev```

### For running frontend tests:
#### Unit Tests:
* ```cd frontend```
* ```npm test```

#### Cypress Tests:
* ```cd frontend```
* ```npx cypress open```
* Select test to run in the GUI
